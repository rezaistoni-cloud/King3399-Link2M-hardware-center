# Firmware Folder

Simpan file firmware lokal di folder ini.

Contoh:

```text
firmware/
├── rk3399_loader_v1.27.126.bin
└── Armbian_26.05.0_rockchip_king3399_noble_6.12.89_server.img
```

File `.img`, `.7z`, `.zip`, dan `.bin` sengaja tidak dimasukkan ke repository karena ukuran besar dan bisa berbeda antar board.
